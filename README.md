# Water NaCl Simulation

### This LAMMPS simulation sets up a 3D system of water with dissolved NaCl, using Lennard-Jones and long-range Coulomb interactions to depict realistic molecular behavior. It runs an NPT simulation to show how the system evolves at constant temperature and pressure and applies constrains to maintain water geometry.
